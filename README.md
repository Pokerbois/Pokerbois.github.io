# Stats & Money Poker  Calculator

This project is a web-based calculator designed to analyze **statistics and money-related data** in a clear and simple way.

The site allows users to input values and instantly see calculated results, making it useful for quick financial estimates and basic statistical analysis.

## Features
- Simple and easy-to-use calculator interface
- Supports money and numerical inputs
- Instant calculations and results
- Built with basic web technologies for fast performance

## Use Cases
- Budgeting and cost estimates
- Basic financial calculations
- Statistics practice and learning
- Classroom or personal use

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Use
Open the website in a browser, enter the required values, and view the calculated results.

## Purpose
This project was created for educational and practical use, focusing on clarity, accessibility, and simplicity.
